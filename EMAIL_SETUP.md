# Email Setup

To set up the email service, follow these steps:

1. Create an account on Resend: https://resend.com
2. Generate an API key: https://resend.com/docs/api-keys
3. Set the `RESEND_API_KEY` environment variable to the generated API key

Example:
```bash
export RESEND_API_KEY=YOUR_API_KEY_HERE
```
Replace `YOUR_API_KEY_HERE` with the actual API key generated on Resend.
